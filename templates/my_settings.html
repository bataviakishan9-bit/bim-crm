How to get your Zoho credentials (first-time setup)

You need 3 things from Zoho API Console: Client ID, Client Secret, and a Refresh Token.

STEP 1 — Open Zoho API Console
  1. Go to api-console.zoho.in
  2. Log in with the Zoho Mail account you send emails from
  3. Click Self Client (or Add Client → Self Client)
STEP 2 — Copy Client ID and Client Secret

Inside your Self Client you will see Client ID and Client Secret — paste them into the fields below.

STEP 3 — Generate and Exchange the Code (all in this page)
  1. Click Generate Code → scope: ZohoMail.messages.ALL,ZohoMail.accounts.READ
  2. Duration: 10 minutes → Create → copy the code
  3. Paste it in the Renew Zoho Token box above → click Save Token
  4. The app will exchange it, save everywhere, and confirm. No manual copy-paste needed.
