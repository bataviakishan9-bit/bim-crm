Zoho Mail API Credentials
How to get your Zoho credentials (step-by-step)

You need 3 things: Client ID, Client Secret, and Refresh Token

STEP 1 — Open Zoho API Console
  1. Go to api-console.zoho.in (India DC) or api-console.zoho.com (US DC)
  2. Log in with your own Zoho Mail account (the one you send emails from)
  3. Click "Self Client" — if you don't see it, click "+ Add Client" and choose Self Client
STEP 2 — Copy Client ID and Client Secret
  1. Once inside your Self Client app, you will see Client ID and Client Secret displayed
  2. Copy both and paste them into the fields below
STEP 3 — Generate Refresh Token
  1. Click the "Generate Code" tab inside Self Client
  2. In the Scope field, paste exactly:
    ZohoMail.messages.ALL,ZohoMail.accounts.READ
  3. Set Time Duration to 10 minutes
  4. Add a description (e.g. "BIM CRM") — optional
  5. Click "Create" — you will see a code appear
  6. Copy that code immediately — it expires in 10 minutes
STEP 4 — Convert code to Refresh Token
  1. Run get_token.py in your BIM_CRM folder:
    python get_token.py
  2. Paste the code when prompted
  3. The script will print your new Refresh Token and save it to .env
  4. Copy the Refresh Token and paste it into the field below
Also update Render
  1. Go to dashboard.render.com → your bim-crm service → Environment tab
  2. Update ZOHO_REFRESH_TOKEN with the new token
  3. Also add/update ZOHO_CLIENT_ID, ZOHO_CLIENT_SECRET, ZOHO_MAIL_ACCOUNT_ID
  4. Click Save Changes — service will redeploy automatically
Where is Account ID? — Log into Zoho Mail → Settings (gear icon) → Developer Space / API → your Account ID is shown there (a long number like 277444000000002002).
