Hunter.io — Email Finder Integration

Hunter.io finds and verifies professional email addresses. Free plan = 25 searches/month.

HUNTER_API_KEY {% if hunter_key %} ✓ Configured {{ hunter_key }} {% else %} Not set {% endif %}
How to get your free Hunter.io API key:
  1. Go to hunter.io → Sign up free
  2. Go to API section in your dashboard
  3. Copy your API key
  4. Open C:\Users\Kishan\BIM_CRM\.env
  5. Paste it: HUNTER_API_KEY=your_key_here
  6. Restart the app

Once set, every lead detail page shows a Hunter.io panel to:

  • Verify the current email (valid / invalid / risky)
  • Find the correct email by name + company domain
  • Search all emails at a company domain
  • One-click update the lead's email when found

How to update credentials
  1. Open C:\Users\Kishan\BIM_CRM\.env in Notepad
  2. Update the values
  3. Save the file
  4. Stop and restart the app: python app.py

.env template:

ZOHO_CLIENT_ID=1000.xxxxxxxxxx
ZOHO_CLIENT_SECRET=xxxxxxxxxx
ZOHO_REFRESH_TOKEN=1000.xxxxxxxxxx.xxxxxxxxxx
ZOHO_DC=in
ZOHO_MAIL_ACCOUNT_ID=1234567890
SECRET_KEY=bim-infra-crm-2025

Don't have credentials yet? Run python C:\Users\Kishan\get_token.py to generate them automatically.
